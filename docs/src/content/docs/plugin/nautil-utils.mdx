### ``archive``
	#### Parameters
	- ``source_path: str`` <Flag variant="templated"/>
		<br/>The path to the file or directory to archive, relative to the artifact.
	- ``output_path: str`` <Flag variant="templated"/>
		<br/>The path to the archive file to create, relative to the artifact.
	- ``archive_format: str = "zip"`` <Flag variant="templated"/>
		<br/>Archive format to create. Supported: ``zip``, ``tar``, ``gztar``, ``bztar``, ``xztar``.
	- ``overwrite: bool = True``
		<br/>Whether to overwrite an existing archive at ``output_path``.

	#### Description
	Creates a zip or tar archive from a file or directory inside the artifact workspace.

	#### Example
	```python
	a = Artifact(env).use(...)
		.archive("dist", "dist.zip", archive_format="zip")
	```

### ``filter``
	#### Parameters
	- ``filter_func: FilePredicate``
		<br/>Predicate called with ``(file_name, file_path, workspace)``; return ``True`` to delete.
	- ``root: str = "."`` <Flag variant="templated"/>
		<br/>Root directory to scan inside the workspace.

	#### Description
	Deletes files and directories that match the given predicate.

    <Aside type="tip">
        look into the [Filter Predicate Builders](#filter-predicate-builders) section below for utilities to build common predicates, such as regex or glob's .ignore-style predicates.
    </Aside>

	#### Example
	```python
	import re
	from nautil_utils.filter import make_regex_predicate

	a = Artifact(env).use(...)
		.filter(make_regex_predicate(re.compile(r".*\\.bak$")), root="build/")
	```
	---

### ``foreach``
	#### Parameters
	- ``filter_func: FilePredicate``
		<br/>Predicate called with ``(file_name, file_path, workspace)``; return ``True`` to process.
	- ``item_action: Callable[..., Any]``
		<br/>Action callable invoked per match; supports multiple signatures.
	- ``root: str = "."``
		<br/>Root directory to scan inside the workspace.
	- ``files: bool = True``
		<br/>Whether to include files.
	- ``dirs: bool = False``
		<br/>Whether to include directories.
	- ``recursive: bool = True``
		<br/>Whether to traverse subdirectories.

	#### Description
	Runs an action for each file or directory that matches a predicate.

	#### Example
	```python
	import os
	import re
	from nautil_utils.filter import make_regex_predicate

	def delete_file(relative_path: str, workspace: str):
		os.remove(os.path.join(workspace, relative_path))

	a = Artifact(env).use(...)
		.foreach(make_regex_predicate(re.compile(r".*\\.tmp$")), delete_file, root="build/")
	```

## Sources

### ``ArtifactSource``
	#### Parameters
	- ``artifact: Artifact``
		<br/>An existing artifact to use as a source.
	- ``root: PathLike = "."``
		<br/>Root path inside the artifact to copy from.
	- ``compressed_format: str | None = None``
		<br/>Optional archive format. Supported: ``zip``, ``tar``, ``gztar``.

	#### Description
	Copies files from another artifact. When ``compressed_format`` is set, the source is archived before copying.

	#### Example
	```python
	from nautil_utils.source import ArtifactSource

	base = Artifact(env).use(...)
	pack = Artifact(env)
	pack.use(ArtifactSource(base, root="dist"), dest=".")
	```
	---

### ``GitSource``
	#### Parameters
	- ``git_url: str``
		<br/>Git repository URL to clone.
	- ``branch: str = "main"``
		<br/>Branch to clone.
	- ``specific_commit: str | None = None``
		<br/>Optional commit hash to checkout after cloning.
	- ``depth: int = 1``
		<br/>Clone depth for shallow clones.

	#### Description
	Clones a Git repository and exposes files as a source.

	#### Example
	```python
	from nautil_utils.source import GitSource

	a = Artifact(env).use(
		GitSource("https://github.com/neylz/nautil-utils"),
		dest="vendor/nautil-utils",
		src_path="nautil_utils"
	)
	```
	---

### ``LocalSource``
	#### Parameters
	- ``path: PathLike``
		<br/>Local directory or file system root path to use as the source.

	#### Description
	Copies files from a local path into the artifact.

	#### Example
	```python
	from nautil_utils.source import LocalSource

    # Copying a directory
	a = Artifact(env).use(
		LocalSource("C:/assets"),
		dest="assets",
		src_path="icons"
	)

    # Copying a single file
    a = Artifact(env).use(
        LocalSource("C:/assets"),
        dest="static/logo.png",
        src_path="icons/logo_512x512.png"
    )
	```

## Filter Predicate Builders

Filters are using predicates. Below are some utilities to build common predicates. You can of course write your own predicate function.

To write a custom predicate, you must input a ``nautil_utils.types.FilePredicate``, which is a callable with the signature
``(file_name: str, file_path: str, workspace: str) -> bool`` (``Callable[[str, str, str], bool]``).
The predicate should return ``True`` for paths that should be matched (e.g. deleted by the ``filter`` action).

### ``make_dotignore_predicate``
	#### Parameters
	- ``ignore_file_name: str``
		<br/>The name (or path) of the .ignore file to read, for example ``.gitignore``.
	- ``workspace_relative: bool = True``
		<br/>When true, resolves ``ignore_file_name`` relative to the workspace root.

	#### Description
	Builds a predicate that follows .ignore-style rules. The returned predicate matches paths that should be ignored.

	#### Example
	```python
	from nautil_utils.filter import make_dotignore_predicate

	ignore_predicate = make_dotignore_predicate(".packignore")

	a = Artifact(env).use(...)
		.filter(ignore_predicate)
	```
	---

### ``make_regex_predicate``
	#### Parameters
	- ``pattern: re.Pattern``
		<br/>Compiled regex used to test relative paths.

	#### Description
	Builds a predicate that matches paths using a regular expression.

	#### Example
	```python
	import re
	from nautil_utils.filter import make_regex_predicate

	a = Artifact(env).use(...)
		.filter(make_regex_predicate(re.compile(r".*\.bak$")))
	```
